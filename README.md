#  FastAPI Doctor & Patient Management API

A simple REST API built using **FastAPI** to manage doctors and patients using in-memory storage.  
This project demonstrates CRUD operations, validation, and API design using FastAPI.

---

##  Features

- Add a new doctor
- Get all doctors
- Get doctor by ID
- Add a new patient
- Get all patients
- Input validation using Pydantic
- Email validation for doctors
- Age validation for patients
- Prevent duplicate IDs
- Simple in-memory data storage

---

## Tech Stack

- Python 3.10+
- FastAPI
- Pydantic
- Uvicorn (ASGI server)
